Recurring Wizard

A 'Generate Recurring' button is displayed on the purchase order.
Recurring Document, Recurring Type and Scheduled action of this particular purchase order is automatically created.
Number of documents - The number of times the document should be recurred.
Interval Unit - The interval when the recurring should occur.
Start Recurring - Activates the generated recurring.
Save - Creates the recurring but doesn't activates it.

Purchase Order

Recurring - Shows the generated recurring.
Document - Links the purchase order that are generated by the recurring of that particular purchase order.
